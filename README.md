### Description

A clone of Hoobank using React JS & Tailwind CSS with Vite.

**Get start âœ¨**

1. Clone the repository:

   ```zsh
   git clone https://github.com/tugu-tmrbtr/hoobank-clone.git
   cd hoobank-clone
   ```

2. Install the dependencies:

   ```zsh
   # Using npm
   npm i
   # Using Yarn
   yarn
   ```

3. Start the development server:

   ```zsh
   # Using npm
   npm run dev
   # Using Yarn
   yarn run dev
   ```

   Now you should be able to see the project running at localhost:5173.

4. Open `./index.html` in your editor and start experimenting!

### Building for production

To build an production version of your app, simply run:

```zsh
# Using npm
npm run build
# Using Yarn
yarn run build
```

After that's done, check out `./dist` to see the optimized output.
